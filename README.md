# Weapon Detection System

This project implements a real-time weapon detection system using YOLOv12n optimized with GWO-ABO.

## Features
- Image Detection
- Video Detection
- Real-Time Webcam Detection

## Tech Stack
- YOLOv12n
- FastAPI
- OpenCV
- Python

## Run

```bash
pip install -r backend/requirements.txt
uvicorn backend.main:app --reload
